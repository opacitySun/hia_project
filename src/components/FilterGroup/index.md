筛选控件，可选择要筛选的项目类型，并动态获取到筛选结果的value值。

## API

### FilterGroup

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| rowTypes    |要筛选的项目类型   |string[] | |
| onChange | 标签选择的回调函数 | Function(resultVals) |  |
| timeSelect | 日期的数据 | Object |  |
| region | 区域的数据 | Object[] |  |
| medicalInstitution | 医疗机构的数据 | Array[] |  |
